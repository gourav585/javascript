<h1>basics of javascript</h1>
<p>JavaScript is a versatile programming language primarily used for front-end web development. Here's a breakdown of the basics:</p>
<ol>
  <li> Variables:</li>
</ol>
<ul>
  <li>Used to store data values.</li>
  <li>Declared using let, const (for constant values), or var (older syntax).</li>
  <li>Example:</li>
</ul>
<div>
  let name = "John";
  <br>
const age = 25;
</div>
<ol>
  <li> Data Types:</li>
</ol>
<ul>
  <li>Primitive Types: </li>
</ul>
<p>String, Number, Boolean, Null, Undefined, Symbol</p>
<ul>
  <li>Objects:</li>
</ul>
<p>Collections of key-value pairs.</p>
<ul>
  <li>Arrays:</li>
</ul>
<p>Ordered collections of data.</p>
<ol>
  <li> Operators:</li>
</ol>
<ul>
  <li>Arithmetic: </li>
</ul>
<p>+, -, *, /, %.</p>
<ul>
  <li>Assignment:</li>
</ul>
<p>=, +=, -=, etc.</p>
<ul>
  <li>Comparison:</li>
</ul>
<p>==, ===, !=, !==, >, <, >=, <=.</p>
<ul>
  <li>Logical:</li>
</ul>
<p>&& (AND), || (OR), ! (NOT).</p>
